<h1>#Usage</h1>
<p>You can use it, like normal component. Copy components folder to your project folder and then import and register in your app.</p>

<pre>
<code>
import player from '@/components/player.vue';

export default {

  name: 'app',

  components:{
    
    player,

  },

};
</code>
</pre>
